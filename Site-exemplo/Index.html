Guia de linguagem MARKDOWN

MARKDOWN é uma linguagem com mais facilidade de leitura, muito semelhante ao HTML


Negrito: **Palavra em negrito**
Italico: *Palavra em italico*
Titulo 1: #titulo aqui.
Titulo 2: ##titulo 2 aqui.


Para fazer uma lista ordenada você precisa colocar o comando numerico acompanhado de um ponto logo em seguida.
1. item 1
1. item 2
1. item 3
Assim a lista será mostrada assim no Github dentro de arquivos MD:

  1. item 1
  2. item 2
  3. item 3


Para que a lista seja feita sem numeradores ou apenas icones de indicação, ela devera ser feita com * ou -
ex:
* item 1
* item 2
* item 3

E será executado assim:

  • item 1
  • item 2
  • item 3
Para que a lista tenha subitens, sera necessário acrescentar 2 espassos antes do item e, para lista de tarefas, colocar colchetes antes dos itens
[]item 1
[]item 2

Para que o tem tenha sido ticado como tarefa cumprida, é só colocar um X entre os colchetes:
[x]item 1
